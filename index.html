OSPFv2 Example

router ospf 1
 router-id 2.2.2.2
 network 172.16.0.0 0.0.255.255 area 0
 passive-interface default
 no passive-interface g0/0
interface g0/0
 ip ospf hello-interval 20
 ip ospf dead-interval 80

Dynamic NAT (R3)

access-list 1 permit 192.168.3.0 0.0.0.255
ip nat pool SITE3_POOL 209.165.17.1 209.165.17.100 netmask 255.255.255.0
ip nat inside source list 1 pool SITE3_POOL overload
interface g0/0
 ip nat inside
interface g0/1
 ip nat outside

Static NAT (R1)

ip nat inside source static 192.168.1.10 200.168.10.1
interface g0/0
 ip nat inside
interface g0/1
 ip nat outside

GRE Tunnel Configuration

interface Tunnel0
 ip address 100.100.100.1 255.255.255.252
 tunnel source [R3-public-IP]
 tunnel destination [R4-public-IP]

IPv6 OSPFv3 Configuration

ipv6 unicast-routing
interface g0/0
 ipv6 address 2001:db8:acad:0010::1/64
 ipv6 address FE80::1 link-local
 ipv6 ospf 1 area 0
 ipv6 ospf hello-interval 20
 ipv6 ospf dead-interval 80
ipv6 router ospf 1
 router-id 1.1.1.1

IPv6 Default Route

ipv6 route ::/0 [next-hop or interface]
ipv6 router ospf 1
 default-information originate
